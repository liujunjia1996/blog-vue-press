## 需求
| id int | fruits varchar |
| ---   |   ------    |
| 1| apple,banana,orange,kiwi,cherry,peach|
| 2| cherry,apple,kiwi|
| 3| kiwi,peach|
| 4| cherry,banana|
| 5| banana |

输入 cherry,banna 可以查出 id 为 1,2,4,5 的数据

## 实现
利用 mysql 的正则匹配功能。
```sql
select id from tbl where fruits regexp  replace("cherry,banna", ",", "|");
```
